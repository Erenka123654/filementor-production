⚙️ Kurulum Rehberi

Filementor Studio'yu iyzico ile canlıya almak için 4 adım.

1

iyzico Sandbox Hesabı Aç

Önce test ortamında dene, hazır olunca canlıya geç.

👉 sandbox-merchant.iyzipay.com adresine git ve üye ol.

Giriş yaptıktan sonra: Ayarlar → Merchant Bilgileri sayfasından API Key ve Secret Key'i kopyala.

📌 Sandbox hesabı ücretsiz ve anlık açılır. Gerçek kart bilgisi gerekmez.
2

Node.js Kur (bir kez)

Bilgisayarında Node.js yoksa indir:

👉 nodejs.org → LTS sürümünü indir ve kur.

Kurulumu doğrulamak için terminali aç ve yaz:

node --version
npm --version

İkisi de versiyon numarası gösteriyorsa hazırsın.

3

Backend'i Başlat

a) ZIP'teki filementor-backend klasörünü masaüstüne çıkar.

b) .env dosyasını bir metin editörüyle aç ve key'lerini gir:

IYZICO_API_KEY=sandbox-xxxxxxxxxxxxxxxxx
IYZICO_SECRET_KEY=sandbox-xxxxxxxxxxxxxxxxx
IYZICO_SANDBOX=true
FRONTEND_URL=*
PORT=4000

c) Terminali aç, klasöre gir ve kur:

# Windows (Komut İstemi veya PowerShell)
cd C:\Users\KULLANICI\Desktop\filementor_studio-backend
npm install

# Mac / Linux
cd ~/Desktop/filementor_studio-backend
npm install

d) Başlat:

node server.js

Şunu görmelisin:

✅ filementor Backend çalışıyor
   http://localhost:4000
   Mod: 🧪 SANDBOX
💡 Backend çalışırken terminali kapatma. Mağaza ile aynı anda açık olmalı.
4

Mağazayı Aç ve Test Et

filementor_studio/index.html dosyasını tarayıcıda aç ve bir ürünü sepete ekle.

"Ödemeye Geç" → bilgileri doldur → kart bölümünde test kartlarından birini kullan:

Kart NoSon KullanmaCVVSonuç
552879000000000812/30123✓ Başarılı
405418000000000712/30123✓ Başarılı
552879000000001612/30123✗ Reddedildi
⚠️ Test kartlarıyla gerçek para çekilmez. Canlıya geçmeden önce tüm akışı sandbox'ta test et.
5

Canlıya Geçiş

Her şey sandbox'ta çalışınca:

1. merchant.iyzipay.com'dan canlı hesap başvurusu yap (firma belgesi gerekir).

2. Onaylandıktan sonra canlı API key'lerini al.

3. .env dosyasını güncelle:

IYZICO_API_KEY=CANLI_API_KEY
IYZICO_SECRET_KEY=CANLI_SECRET_KEY
IYZICO_SANDBOX=false

4. node server.js → artık gerçek ödeme alabilirsin!

📦 Sunucuya yüklemek istersen Railway.app veya Render.com'da ücretsiz Node.js hosting var.
